# Check if the Same Case

`Strings` `Validation`

Create a function that returns `true` if an input string contains only uppercase or only lowercase letters.

### Examples

```js
sameCase("hello") ➞ true

sameCase("HELLO") ➞ true

sameCase("Hello") ➞ false

sameCase("ketcHUp") ➞ false
```
