# Are the Numbers Equal?

`Conditions` `Language Fundamentals` `Logic`



Create a function that takes two integers and checks if they are equal.

### Examples

```js
isEqual(5, 6) ➞ false

isEqual(1, 1) ➞ true

isEqual("1", 1) ➞ false
```

### Notes

If there is a string then it should return `false`.
